# TransLinks

The TransLinks component allows interpolation of links and text from the internationalization data into a React component

This component is passed two parameters:

- i18nTextKey -- This maps to a key in the localization data to a string that may have styles, but has placeholders for links
- i18nLinkKey -- This maps to a list of URLs, to be interpolated into the above

## A simple example

These are the values of the internationalization strings in the file
`public/locales/en/common.json`

<TransDescription {...Default.args} />

</Canvas>
